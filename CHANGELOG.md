# 변경 이력

TESTER의 주요 업데이트 내역입니다.

## [0.7.1] - 2026-02-10

### 🐛 버그 수정
- **History API 응답 오류 수정**: `/api/history/` 엔드포인트에서 발생하던 ResponseValidationError 해결
  - Pydantic 모델 타입을 실제 데이터와 일치시킴 (`UUID`, `datetime` 타입 정합성)
  - `from_attributes = True` 설정 추가로 ORM 객체 자동 변환 활성화
  - 26개 검증 오류가 모두 해결됨

## [0.7.0] - 2026-02-09

### ✨ 새로운 기능
- **상세 서비스 상태 확인**: 시스템 상태를 더 자세히 확인할 수 있습니다 (응답 속도, 연결 상태 등)

### 🚀 성능 개선
- **더 빠른 응답**: 반복적인 요청에 대한 응답 속도가 크게 향상되었습니다
- **메모리 최적화**: 서버 자원을 더 효율적으로 사용합니다

### 🛡️ 안정성 향상
- **데이터 보호**: 생성 이력이 확실하게 저장되어 데이터 손실을 방지합니다
- **연결 안정화**: Redis 연결이 더 안정적으로 유지됩니다
- **안전한 종료**: 서버 종료 시 모든 작업이 깔끔하게 정리됩니다

### 🌏 사용 편의성
- **한글 오류 메시지**: 대부분의 오류 메시지가 한글로 표시됩니다
- **더 명확한 피드백**: 작업 실패 시 원인을 더 쉽게 파악할 수 있습니다
