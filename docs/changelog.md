# 변경 기록 (Changelog)

## [2026-02-19]
- (New) **토큰 시스템 개편**: 광고 보상 시스템을 제거하고, Ko-fi 후원($5 -> 200토큰) 연동으로 대체했습니다.
- (New) **보너스 상향**: 신규 가입 보너스(50토큰) 및 일일 로그인 보너스(30토큰)가 상향 조정되었습니다.
- (Security) **보안 강화**: 토큰 지급 로직의 동시성 문제(Race Condition)를 해결하여 중복 지급을 원천 차단했습니다.
- (UI) **인터페이스 수정**: 사용자 요청에 따라 후원 버튼 디자인을 변경하고 불필요한 아이소를 제거했습니다.

## [2026-02-18]
- (New) **Ko-fi 후원 연동**: 광고 시청 대신 Ko-fi를 통해 후원하고 토큰을 충전할 수 있는 기능을 추가했습니다.
- (Remove) **광고 보상 제거**: 사용자 경험 개선을 위해 광고 기반 토큰 지급 기능을 제거했습니다.

## [2026-02-15]
- (New) **보안 강화**: 시스템 안전성을 위해 중요 데이터 접근 방식을 개선하고, 실행 환경의 보안 설정을 업데이트했습니다.
- (Fix) **성능 최적화**: 서비스 실행 속도를 높이기 위해 내부 연결 방식을 개선했습니다.
